# Cyberpunk GitHub Pages Personal Homepage

A static GitHub Pages personal homepage inspired by a desktop interface: neon blue/pink cyberpunk style, clickable folder icons, and draggable popup windows.

## Files

- `index.html` — page structure and English profile content
- `style.css` — cyberpunk visual design
- `script.js` — folder opening, draggable windows, and clock

## Customize

Edit `index.html` and replace:

- `YOUR ENGLISH NAME`
- `YOUR CHINESE NAME`
- the one-line tagline
- research interests
- About Me, Education, Interest, and Contact content
- contact links

## Deploy to GitHub Pages

1. Create a repository named `yourusername.github.io`.
2. Upload `index.html`, `style.css`, `script.js`, and `README.md` to the repository root.
3. Go to **Settings → Pages**.
4. Set source to the main branch and root folder.
5. Visit `https://yourusername.github.io`.

## Pixel Font Update

The site now uses pixel-style cyberpunk typography across the whole interface via Google Fonts (`Press Start 2P` for display UI text and `VT323` for readable body text). No font files are bundled in this folder. If Google Fonts cannot load, the page falls back to monospace system fonts.
